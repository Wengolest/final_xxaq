企业信息安全策略 v3.2

一、系统访问控制
1. 所有生产服务器通过堡垒机登录, 禁止直接 SSH 连接。
2. 密码长度不少于 12 位, 包含大小写字母、数字和特殊字符。
3. 每 90 天强制更换密码, 不得重复使用最近 5 次密码。
4. 多因素认证 (MFA) 对所有管理员账户强制启用。

二、网络安全
1. 默认拒绝所有入站流量, 仅开放业务必要的端口 (80/443)。
2. 内部网络按业务域划分 VLAN, 域间通过防火墙隔离。
3. 所有 API 接口强制使用 HTTPS, 证书由内部 CA 签发。

三、数据保护
1. 生产数据库每日凌晨 2:00 自动全量备份, 保留 30 天。
2. 敏感数据 (身份证号、银行卡号、手机号) 必须加密存储。
3. 备份数据异地存储, 距离主数据中心至少 500 公里。

四、审计与合规
1. 所有管理员操作记录审计日志, 保留不少于 180 天。
2. 每季度进行一次安全漏洞扫描, 高危漏洞 48 小时内修复。
3. 每年进行一次渗透测试, 由第三方安全公司执行。
