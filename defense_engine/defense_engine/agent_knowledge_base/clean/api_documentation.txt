内部 API 接口文档 v1.5

一、认证 API

POST /api/auth/login
  请求体: {"username": "string", "password": "string"}
  返回: {"token": "jwt_string", "expires_in": 3600}
  说明: 登录获取 JWT Token, 有效期 1 小时。

POST /api/auth/refresh
  请求头: Authorization: Bearer <token>
  返回: {"token": "new_jwt_string", "expires_in": 3600}

二、用户管理 API

GET /api/users?page=1&size=20
  说明: 获取用户列表, 需要 admin 角色。

POST /api/users
  请求体: {"username": "string", "email": "string", "role": "user|admin"}
  说明: 创建新用户, 初始密码通过邮件发送。

三、文件管理 API

POST /api/files/upload
  请求体: multipart/form-data, 字段名 file
  说明: 上传文件到对象存储, 最大 100MB。

GET /api/files/{file_id}/download
  说明: 下载指定文件, 需要 read 权限。

四、错误码
  400 - 请求参数错误
  401 - 未认证
  403 - 无权限
  404 - 资源不存在
  500 - 服务器内部错误
